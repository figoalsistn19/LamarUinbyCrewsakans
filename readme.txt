**Cara menjalankan source code menggunakan android studio**

Tools yang digunakan :
- Android Studio
- Emulator / Device android

1. Install android studio dengan mengakses link : https://developer.android.com/studio/
2. Ketika pertama kali membuka Android Studio dan muncul window “Welcome to Android Studio”, klik “Open an existing Android Studio Project”
3. Pilih project android studio dari file zip yang sudah di ekstrak
4. Sebelum menjalankannya, kita perlu menyiapkan emulator atau real device android. Pada tutorial ini akan dijelaskan menggunakan real device android.
5. Siapkan real device android dan aktifkan developer mode. Berikut adalah langkah-langkah untuk mengaktifkan Developer Mode pada device :
	1) Settings → About Phone →  Tekan Build Number 7 kali.
	2) Settings →  Developer Options → Aktifkan USB debugging.
	3) Hubungkan device dengan komputer via USB. 
6. Setelah siap, pilih "Run" pada toolbar di Android Studio


NOTE
Jika aplikasi mengalami error pada firebase, silakan ikuti langkah berikut :
1. Pada menubar di bagian paling atas Android Studio silakan pilih Tools → Firebase
2. Pilih Cloud Firestore → Get Started with Cloud Firestore
3. Klik "Connect to Firebase"
